# Sample Data Description

## Overview
This folder contains synthetic retail data modeled after typical retail and service business scenarios.

## Datasets

### 1. sales_data.csv
Transaction-level sales data including:
- Transaction ID
- Date and time
- Customer ID
- Product details
- Quantity and revenue
- Store location

### 2. customer_data.csv
Customer demographic and behavioral data:
- Customer ID
- Demographics
- Registration date
- Customer segment
- Lifetime value

## Data Generation
All data is synthetically generated for demonstration purposes and does not contain any real customer information.

## Usage
These datasets are used across SQL analysis, Python notebooks, and ETL pipeline demonstrations.
